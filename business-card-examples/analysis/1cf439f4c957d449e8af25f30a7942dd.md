### Chevron Decorative System — THREE LAYERS

The left ~50% of the front features a layered chevron/V-shape system creating depth through **three distinct brightness zones**:

#### Layer 1 — LIGHTER CHEVRON BANDS (`#324154`)
- **Color**: `#324154` (RGB 50, 65, 84) — muted steel-blue
- **Effective opacity**: ~30% white overlay on base (computed: base #1E2633 → overlay result #324154)
- **Band width**: ~12% of card width (consistent throughout)
- **Angle**: ~43° from horizontal (nearly 45°, slope ≈ 0.94)
- **Shape**: Two stacked V/chevron shapes pointing **RIGHT** (">")

**Chevron V1 (upper):**
- **Top arm**: starts at card (0%, 5%) — left edge near top
- **Arm trajectory**: moves diagonally down-right, maintaining 12% band width
- **Vertex**: at approximately card (22%, 40%) — the rightmost point of the V
- **Bottom arm**: returns diagonally down-left from vertex
- **Bottom of V1**: reaches card (0%, 43%) at left edge

**Chevron V2 (lower, parallel repeat):**
- **Top arm**: starts at card (0%, 43%) — immediately below V1
- **Same 12% band width, same ~43° angle**
- **Vertex**: at approximately card (22%, 78%)
- **Bottom arm**: returns to card (18%, 95%) near bottom-right of chevron zone

**Canvas2D Path for one V-arm (band):**
Each band is a parallelogram. For V1 top arm:
```

lineTo(12%, 5%)          // top-right of band at top
lineTo(22%, 40%)         // bottom-right at vertex
lineTo(10%, 40%)         // bottom-left at vertex
```

**Dark V1 (upper):**

- **Vertex**: approximately at card (23%, 54%)
- **Splits/overlaps with V2 below**

**Dark V2 (lower):**
- **Traces from** (~46%, 54%) back down to (~23%, 92%)
- **Same angle and width as lighter chevrons**

#### Layer 3 — BASE BACKGROUND
- **Color**: `#1E2633` — visible between and around all chevron bands
- **Acts as the "neutral" zone** — neither lighter nor darker overlays

#### LAYERING ORDER (bottom to top):
1. `#1E2633` base fill (entire card)

## DESIGN DNA & DEPTH ANALYSIS

### How the Chevrons Create Depth

1. **Three-tone layering on dark background**:
   - Base (`#1E2633`, brightness 39) = mid-depth
   - Dark overlay (`#1A202A`, brightness 33) = recessed/shadow layer
   - Light overlay (`#324154`, brightness 66) = raised/highlight layer
   - This creates a **bas-relief** effect — subtle 3D dimensionality on a flat surface

2. **V-shape repetition creates rhythm**:
   - Two V-shapes stacked vertically = visual heartbeat
   - Each V points RIGHT (">") toward the text zone, creating **directional energy** funneling the eye to the company name

3. **Containment to left 50%**:
   - Chevrons never cross the midpoint
   - Creates a clear **split layout**: decorative left | informational right
   - The gap between chevrons and text is generous (~34% of card width) — this breathing room is essential to the premium feel

4. **Front↔Back mirroring**:
   - Front: chevrons on LEFT, pointing RIGHT
   - Back: chevrons on RIGHT, pointing LEFT
   - Creates **visual bookends** — when card is flipped, the pattern appears to wrap around
   - Back chevrons are much more subtle (only ~8% contrast vs ~45% on front)

5. **Opacity progression**:
   - Front light chevrons: ~30% opacity feel (bold, visible)
   - Front dark chevrons: ~15% opacity feel (subliminal, felt not seen)
   - Back chevrons: ~8% opacity feel (whisper-level, textural)
   - This creates a **front-to-back depth gradient** — bold → subliminal → whisper

### Typography Hierarchy
1. **COMPANY** (front, 11.4% height, white on dark) — hero element
2. **JONATHAN DOE** (back, 5.1% height, dark on light) — primary info
3. **Tagline** (front, 4.3% height, muted) — secondary identity
4. **Graphic Designer** (back, 4.1% height, gray) — secondary info
5. **Contact lines** (back, 2.5% height, blue-gray) — tertiary info
6. **yourwebsite.com** (front, 2.7% height, gray) — subtle CTA

### Design Personality
- **Mood**: Corporate-professional with creative undertone
- **Suitable for**: Design agencies, tech companies, architects, consultants
- **Key adjective**: **Structured sophistication** — geometric precision + restrained palette
- **Avoids**: Color, curves, playfulness — pure angular minimalism

### Chevron Path Template (Canvas2D)
```javascript
function drawChevronV(ctx, cardW, cardH, tipX, tipY, vertexX, vertexY, bandWidth, color) {
  ctx.fillStyle = color;
  // Top arm of V
  ctx.beginPath();
  ctx.moveTo(tipX, tipY);
  ctx.lineTo(tipX + bandWidth, tipY);
  ctx.lineTo(vertexX + bandWidth, vertexY);
  ctx.lineTo(vertexX, vertexY);
  ctx.closePath();
  ctx.fill();

  // Bottom arm of V (mirror of top)
  ctx.beginPath();
  ctx.moveTo(vertexX, vertexY);
  ctx.lineTo(vertexX + bandWidth, vertexY);
  ctx.lineTo(tipX + bandWidth, tipY + 2 * (vertexY - tipY));
  ctx.lineTo(tipX, tipY + 2 * (vertexY - tipY));
  ctx.closePath();
  ctx.fill();
}

// Front — Light chevron V1
drawChevronV(ctx, W, H,
  0, H*0.05,              // tip at top-left
  W*0.22, H*0.40,         // vertex at right
  W*0.12,                 // band width
  '#324154'
);

// Front — Light chevron V2
drawChevronV(ctx, W, H,
  0, H*0.43,
  W*0.22, H*0.78,
  W*0.12,
  '#324154'
);

// Front — Dark chevron V1
drawChevronV(ctx, W, H,
  W*0.46, H*0.10,         // tip at right
  W*0.23, H*0.54,         // vertex at left (reversed!)
  -W*0.10,                // negative width = reversed direction
  '#1A202A'
);
```
